◆ list_iter()

struct list_node * list_iter ( struct list_node * p_node)
extern

Walk the list returning one item for each call.

Parameters
p_nodethe list head or current item in the list

The first time called, the parameter should be the list head. The function will return the next item in the list. With each successive call, the parameter should be item that was returned from the previous call. When the end of the list is reached, NULL will be returned.

Returns
the next item in the list or NULL.
