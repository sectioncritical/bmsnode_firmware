Turn on test mode for specified test function.

Parameters
testmodethe test function to activate
val0function dependent parameter (see command docs)
val1function dependent parameter

Test mode will be started for the function specified by testmode parameter. Only one test function can be active at a time. Other BMSNode features cannot be used while a test function is active. If not stopped by command, test mode will automatically time out after 60 seconds (assuming testmode_run() continues to be called.

Once test mode is turned on, testmode_run() must be called frequently to monitor and run the test function.

◆ testmode_run()

testmode_status_t testmode_run ( void )
extern

Run test mode function and monitoring.

Returns
the current test mode

When test mode is turned on with testmode_on(), the client must call this function frequently from the main loop. If it is not called then it is possible that a test function, such as turning on the shunt load, could be left unmonitored and cause physical damage.
